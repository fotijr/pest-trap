# Pest Trap
Project to detect pests with an ultrasonic sensor and trap them.

## Hardware
- Arduino Nano Every ([pinout](./nano-every-pinout.png))
- HC-SR04 Ultrasonic Sonar Distance Sensor ([Adafruit](https://www.adafruit.com/product/3942))
